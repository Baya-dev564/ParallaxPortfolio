ContactsAPI

Gestionnaire de contacts avec Node.js et MongoDB

Voir la démo Code GitHub

Vue d'ensemble

ContactsAPI est une application full-stack moderne développée avec Node.js et MongoDB. J'ai créé cette application pour découvrir l'écosystème JavaScript côté serveur et les bases de données NoSQL, tout en implémentant des fonctionnalités de recherche avancée.

Technologies utilisées

Node.js

Runtime JavaScript pour le développement backend

Express.js

Framework web minimaliste et flexible

MongoDB

Base de données NoSQL avec documents JSON

Docker

Conteneurisation pour un environnement reproductible

Fonctionnalités principales

CRUD complet des contacts

Gestion complète avec validation côté serveur et client

Recherche en temps réel

Système de recherche instantané sur tous les champs

Tri multi-critères

Tri par nom, prénom, email ou date de création

Statistiques dynamiques

Tableaux de bord avec métriques en temps réel

Interface moderne

Design responsive avec effets glassmorphism

Architecture scalable

Séparation claire front-end/back-end pour la scalabilité

Défis techniques rencontrés

Transition vers NoSQL

Apprentissage de MongoDB après avoir travaillé exclusivement avec des bases SQL. Adaptation à un modèle de données flexible sans schéma strict.

Programmation asynchrone

Maîtrise d'async/await et de la gestion d'erreurs en JavaScript pour éviter les callback hell et garantir la robustesse.

Containerisation Docker

Première expérience avec Docker pour créer un environnement de développement reproductible et faciliter le déploiement.

Ce que j'ai appris

  • Développement d'APIs avec Node.js et Express.js
  • Gestion des bases de données NoSQL avec MongoDB
  • Utilisation de Docker pour la conteneurisation
  • Programmation asynchrone JavaScript avancée
  • Création d'interfaces utilisateur interactives sans framework
  • Implémentation de fonctionnalités de recherche performantes
  • Architecture full-stack moderne et maintenable

Évolutions prévues

Authentification utilisateur, export CSV/PDF, catégories de contacts, notifications en temps réel
