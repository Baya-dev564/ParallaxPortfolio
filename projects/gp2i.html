GP2i

Site vitrine pour une agence de conseil en création web et communication digitale

Voir la démo Code GitHub

Vue d'ensemble

GP2i est le site vitrine d'une agence de conseil basée à Montpellier, spécialisée dans la création de sites web professionnels et la gestion des réseaux sociaux. J'ai développé ce projet avec une architecture PHP MVC complète, une base de données MySQL, un système de formulaire de contact sécurisé et une gestion rigoureuse du consentement RGPD. Le tout est déployé via Docker pour garantir la portabilité de l'environnement.

Technologies utilisées

PHP 8 (MVC)

Architecture Modèle-Vue-Contrôleur avec routeur maison et autoloading PSR-4

MySQL

Base de données relationnelle pour les contacts et consentements RGPD

Docker

Conteneurisation de l'environnement PHP, MySQL et Apache

Composer

Gestion des dépendances PHP et autoloading PSR-4

Fonctionnalités principales

Formulaire de contact sécurisé

Validation côté serveur, protection CSRF, rate limiting et envoi d'email via PHPMailer

Gestion RGPD complète

Bannière de consentement cookies, enregistrement des consentements en base de données

Sécurité renforcée

Protection XSS, injection SQL, headers de sécurité HTTP et sanitisation des entrées

Routeur PHP maison

Système de routing GET/POST vers les controllers sans framework externe

Présentation des services

Création de sites web, gestion réseaux sociaux, SEO, maintenance et support

Interface responsive

Design moderne et optimisé pour tous les appareils

Défis techniques rencontrés

Architecture MVC sans framework

Conception d'un routeur et d'un système MVC from scratch en PHP pur, avec autoloading PSR-4 via Composer, sans s'appuyer sur Laravel ou Symfony.

Conformité RGPD

Mise en place d'un système complet de gestion du consentement cookies, stockage en base avec versioning de la politique de confidentialité.

Sécurisation du formulaire

Implémentation de plusieurs couches de sécurité : tokens CSRF, rate limiting par session, validation et sanitisation de toutes les entrées utilisateur.

Ce que j'ai appris

  • Conception d'une architecture MVC PHP sans framework
  • Mise en place d'un système de routing maison GET/POST
  • Gestion complète de la sécurité web (CSRF, XSS, injections SQL)
  • Conformité RGPD : consentement, politique de confidentialité, stockage
  • Configuration et utilisation de Docker pour PHP/MySQL
  • Envoi d'emails transactionnels sécurisés avec PHPMailer
  • Autoloading PSR-4 et gestion des dépendances avec Composer

Évolutions prévues

Espace client sécurisé, tableau de bord admin pour gérer les demandes de contact, intégration d'un CMS léger, analytics conformes RGPD
