NOI'ART

Plateforme de valorisation de la création humaine face à l'IA

Voir la démo Code GitHub

Vue d'ensemble

NOI'ART est une plateforme pensée pour valoriser la création humaine dans un monde où les images générées par IA se multiplient. Elle naît d'un constat simple : les artistes, illustrateurs et graphistes doivent pouvoir montrer leur travail sans être noyés dans des contenus générés par IA. La plateforme met en place des outils pour distinguer le travail humain et redonner de la valeur à la relation entre créateurs et recruteurs.

Technologies utilisées

HTML5

Structure sémantique et accessibilité des contenus

CSS3

Design immersif, animations et mise en page responsive

Fonctionnalités principales

Profil artiste

Espace dédié à chaque créateur pour exposer son travail et son univers artistique

Badge "100% Humain"

Système de distinction visuelle pour certifier les œuvres créées par des humains

Mise en relation

Interface de contact entre artistes et recruteurs ou clients potentiels

Galerie de projets

Présentation visuelle soignée des portfolios d'artistes

Design immersif

Interface artistique et animée pour mettre en valeur les créations

Interface responsive

Expérience optimale sur tous les appareils, du mobile au desktop

Défis techniques rencontrés

Design artistique en CSS pur

Créer une interface visuellement forte et immersive sans framework, en travaillant finement les animations, dégradés et transitions CSS.

Accessibilité et contrastes

Maintenir un design artistique tout en respectant les standards d'accessibilité (contrastes, lisibilité, navigation clavier).

Responsive sans framework

Adapter une mise en page complexe et artistique à toutes les tailles d'écran uniquement avec CSS Grid et Flexbox.

Ce que j'ai appris

  • Conception d'interfaces orientées expérience utilisateur et émotion
  • Maîtrise avancée des animations CSS (keyframes, transitions)
  • Structuration sémantique HTML5 pour l'accessibilité
  • Responsive design sans framework CSS
  • Réflexion UX/UI sur un sujet à fort enjeu sociétal (IA vs création humaine)
  • Importance du storytelling visuel dans un projet web

Évolutions prévues

Système d'authentification artiste, vérification d'œuvres humaines via metadata, messagerie intégrée recruteur/artiste, moteur de recherche par style artistique
