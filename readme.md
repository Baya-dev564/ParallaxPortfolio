# Portfolio Parallax - Développeur Web

Un portfolio moderne avec effets parallax développé en HTML, CSS et JavaScript vanilla.

## Aperçu

Ce portfolio présente mes projets de développement web avec une interface moderne et des animations fluides. Il met en avant mes compétences en front-end et back-end à travers quatre projets principaux.

## Technologies utilisées

- **HTML5** - Structure sémantique moderne
- **CSS3** - Animations avancées, Grid, Flexbox, Variables CSS
- **JavaScript ES6+** - Interactions dynamiques et effets parallax
- **Design responsive** - Optimisé pour tous les appareils

## Projets présentés

### TaskMaster
Application de gestion de tâches avec interface interactive
- HTML/CSS/JavaScript pur
- LocalStorage pour la persistance
- Interface responsive avec animations

### BlogAPI
API REST complète pour système de blog
- PHP 8 avec architecture MVC
- MySQL avec requêtes optimisées
- Authentification JWT

### ContactsAPI  
Gestionnaire de contacts moderne
- Node.js avec Express.js
- MongoDB pour le stockage NoSQL
- Recherche avancée en temps réel

### EcoRide
Application de covoiturage écologique
- Géolocalisation avec API Google Maps
- Système de réservation complet
- Calcul d'empreinte carbone

## Fonctionnalités

- Effet parallax fluide sur la page d'accueil
- Animations au scroll pour les compétences
- Navigation responsive avec menu hamburger
- Pages projet détaillées avec défis techniques
- Design glassmorphism moderne
